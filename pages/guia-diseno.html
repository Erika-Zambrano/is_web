8. Cómo entregar

Opción A — Documento

Completar esta misma página con los valores y enviarla. Se aplica cada cambio directamente al código.

Opción B — Tokens JSON

Entregar un archivo tokens.json con los pares variable→valor.

Opción C — Figma

Compartir el archivo de diseño. Se extraen los valores de los estilos globales.

Ejemplo tokens.json

{
  "--bg": "#ffffff",
  "--accent": "#E84B7A",
  "--accent-hover": "#f06090",
  "--font-head": "Syne",
  "--font-body": "DM Sans"
}
Lo que NO hace falta especificar: márgenes, espaciados, bordes redondeados, animaciones y layout de las secciones — están definidos en el código y no cambian.
